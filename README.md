### [Semantic Versioning](https://semver.org/)

このプロジェクトは、セマンティックバージョニングを使用しております。

基本的に下記の項目を守っていただければ問題ありません。

#### メジャーバージョン

下位互換性のないバージョンアップが行われる場合のパッチ

- メジャーバージョンが0の段階は破壊的変更をマイナーやパッチでも加えても良い

#### マイナーバージョン

新規の下位互換性がある場合アップデートを行う

#### パッチバージョン

APIに影響しないバグのフィックス
